JAVASCRIPT CURRICULUM

Explore Javascript for Web

Students will study about how to use javascript with thier website. Javascript is used to interact, manipulate and add the logic to your website. In this course we will cover the basic syntax of javascript programming, fundamental concept of programming such as condition, loop and function, DOM manipulation, event, asynchronous, request data from api and database. Then we will explore some popular javascript libraries to make website better such as bootstrap, jquery, animation on scroll, swiper, leaflet etc.

Term1

Main Topics

  • Introduction to Javascript
  • Fundamental of Programming: condition, loop, function
  • Interact and manipulate website with DOM and event listener

Timeline

For detail lesson please check document folder: Lesson Plan Note for Term1
